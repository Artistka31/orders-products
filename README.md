# Orders & Products SPA

Single-page application for managing orders, groups, and products.

---

## Technologies

- **React 18**
- **Vite 7.2.6** — build tool and dev server
- **Zustand 5.0.9** — global state management
- **React Router 6.16.0** — application routing
- **React Transition Group 4.4.5** — animations
- **Animate.css 4.1.1** — CSS animations
- **Socket.IO Client 4.8.1** — WebSocket connection

---

## Project Setup

### Backend (WebSocket Server)

1. Go to the server folder:

```bash
cd server

Run server:
node server.js

Server will run on: http://localhost:3001
```

### Frontend

1. Install dependencies:

```bash
npm install

Run frontend (Vite dev server):
npm run dev

Open in Browser
http://localhost:5173
```
